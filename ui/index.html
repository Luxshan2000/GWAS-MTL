About the Project

Project Overview

This project implements a multitask learning model for predicting disease risk based on GWAS data. The model uses shared layers with an attention-based pooling mechanism to enhance genetic feature extraction across multiple diseases.

Methodology

We used real cardiovascular disease GWAS data from the CARDIoGRAMplusC4D consortium and created derived datasets for Type 2 Diabetes and Cancer to demonstrate the multitask learning approach.

Model Architecture

The model consists of shared feature extraction layers, an attention-based pooling mechanism, and disease-specific output layers. This architecture allows the model to leverage shared genetic patterns while still capturing disease-specific associations.

Future Work

Future improvements could include using real GWAS datasets for all diseases, incorporating larger SNP sets, and adding population stratification controls for more robust predictions.
